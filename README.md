#  Student Attendance Management System

A Python GUI app using **Tkinter**, **NumPy**, and **Matplotlib** to record and visualize student attendance.

### Features

* Load students from CSV
* Mark Present/Absent
* Save daily attendance (CSV)
* View attendance graph
### Tech Stack
Python • Tkinter • NumPy • Matplotlib • CSV
###  Run
```bash
pip install numpy matplotlib
python main.py
```
###  CSV Format
```
Roll No,Name
1,John Doe
2,Jane Smith
```
###  Future Scope
Database integration • Report generation • Face/QR attendance
